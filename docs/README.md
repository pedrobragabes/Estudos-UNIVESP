# Documentacao do Projeto

Esta pasta centraliza os documentos de planejamento e manutencao do repositorio.

## Arquivos

- `ROADMAP.md`: plano de evolucao funcional e tecnica do projeto.

## Objetivo da documentacao

- Organizar os proximos passos do app de estudos.
- Facilitar manutencao e contribuicoes futuras.
- Registrar direcionamento de produto e prioridades.

## Boas praticas

- Manter os arquivos curtos e objetivos.
- Atualizar o roadmap a cada ciclo de melhorias.
- Registrar apenas informacoes que ajudam na execucao.
