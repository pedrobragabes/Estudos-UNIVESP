# Roadmap - Estudos UNIVESP

Planejamento de evolucao do projeto para 2026.

## Estado atual (base pronta)

- [x] Flashcards por materia
- [x] Quiz com temporizador
- [x] Simulado com questoes mistas
- [x] Tutor IA integrado ao Gemini
- [x] Persistencia de progresso local

## Fase 1 - Consolidacao de conteudo

- [ ] Revisar e padronizar todos os cards existentes
- [ ] Aumentar banco de questoes por disciplina
- [ ] Incluir explicacao de gabarito para cada questao
- [ ] Adicionar mais exemplos de erros comuns

## Fase 2 - Melhorias de estudo guiado

- [ ] Trilha de revisao por dificuldade
- [ ] Modo de revisao de erros recorrentes
- [ ] Simulado adaptativo baseado em desempenho
- [ ] Estatisticas por materia e por periodo

## Fase 3 - Experiencia e produtividade

- [ ] Melhorias visuais e acessibilidade
- [ ] Atalhos de teclado em mais modulos
- [ ] Exportacao de resumo de desempenho
- [ ] Melhor suporte mobile

## Fase 4 - Publicacao e colaboracao

- [ ] Pipeline de CI para build e checagens
- [ ] Templates de issue e pull request
- [ ] Guia para contribuidores
- [ ] Versionamento por releases no GitHub

## Fase 5 - Evolucao com IA

- [ ] Prompts especializados por materia
- [ ] Sugestao de plano semanal de estudos
- [ ] Feedback automatico por topico de erro
- [ ] Banco de explicacoes com foco em provas

## Observacoes

- Prioridade imediata: estabilidade, seguranca e qualidade do conteudo.
- Novas features entram somente apos revisao da base atual.
